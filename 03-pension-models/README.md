# Pension Models

This folder contains selected Excel-based pension modelling projects. The models were created to analyse different pension system logics, including individual account accumulation, annuity conversion and pension point accumulation.

The purpose of these projects is to demonstrate long-term financial modelling, Excel-based simulation, actuarial thinking and sensitivity analysis in the context of pension systems.

## Main focus areas

- Pension modelling
- Long-term financial projections
- Excel-based simulation
- Individual account accumulation
- Life annuity calculation
- Pension point system modelling
- Wage path assumptions
- Sensitivity analysis
- Retirement income estimation

## Model 1: Deterministic Individual Account and Life Annuity Pension Model

This model follows an individual account logic. During the active working period, contributions accumulate on an individual account with assumed wage growth and investment returns. At retirement, the accumulated pension wealth is converted into an initial pension payment using an annuity factor based on mortality assumptions.

Main elements:

- Starting work age
- Retirement age
- Career-based wage path
- Monthly contributions
- Assumed nominal return
- Accumulated pension wealth
- Life annuity factor
- Initial annual and monthly pension
- Sensitivity analysis

## Model 2: Pension Point System Model

This model follows a pension point system logic. In each year, the individual receives pension points based on the ratio of individual annual gross earnings to national average gross earnings. The total accumulated points are then converted into an initial pension using a calibrated point value.

Main elements:

- Annual pension point accumulation
- Individual wage path
- National average wage path
- Total accumulated pension points
- Calibrated point value
- Inflation-indexed point value
- Initial pension estimation
- Sensitivity analysis
