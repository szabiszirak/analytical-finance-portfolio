# Excel VBA Automation Projects

This folder contains selected Excel and VBA-based projects related to financial modelling, stock price analysis, return calculations, simulations and portfolio optimisation.

The projects were created in Microsoft Excel using VBA macros, formulas, charts and automated workflows. The main purpose of these files is to demonstrate practical Excel automation, financial data processing and model-building skills.

## Main focus areas

- Excel automation
- VBA macros
- Stock price and return analysis
- Log return calculations
- Financial simulations
- Correlation analysis
- Portfolio optimisation
- Order book and futures contract modelling
- Data visualisation in Excel
