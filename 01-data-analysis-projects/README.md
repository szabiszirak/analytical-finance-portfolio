# Data Analysis Projects

This folder contains selected Hungarian-language statistical and econometric analysis projects. The projects are based on Excel datasets, R scripts and written reports.

The analyses cover consumer price time series, exchange rate modelling, stock market volatility and transport-related time series modelling.

## Main focus areas

- Time series analysis
- Trend fitting
- Seasonality analysis
- Stationarity testing
- ARIMA and ARMA modelling
- GARCH and GO-GARCH modelling
- VAR modelling
- Granger causality
- Data visualisation in R
- Economic interpretation of statistical results
