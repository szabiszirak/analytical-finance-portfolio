# Exchange Rate VBA Automation Tool

## Project overview

This project is an Excel-based VBA tool designed to automate exchange rate-related calculations and data processing tasks.

## Objective

The aim of the project was to reduce manual Excel work by using VBA macros for repetitive financial data processing steps.

## Main features

- Automated data handling in Excel
- Exchange rate-related calculations
- VBA-based workflow automation
- Structured input and output sheets
- Reduced manual copying and formatting

## Tools used

- Microsoft Excel
- VBA
- Financial data processing

## Files

- `excel/`: Excel workbook containing the tool
- `vba_code/`: exported VBA modules
- `screenshots/`: screenshots showing the structure and output of the tool

## Notes

The uploaded version contains anonymised or sample data only.
